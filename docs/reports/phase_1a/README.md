# Phase 1a Reports

Phase 1a documents the first runnable target-weight-first strategy system:

- S0 baseline battery.
- S1 quant core for Track 1 and Track 2.
- S2/S3 bounded structured LLM strategies with off/manual/api modes.
- S4 Track 2 event-to-exposure overlay with Track 1 fallback.

Generated run artifacts stay under `.var/runs/`; manual LLM prompts and responses stay under `.var/manual_llm/`.
